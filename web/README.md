### 💻 About

**Brev.ly** is a responsive URL shortener with a user dashboard to manage created links.

It shortens URLs with persistent storage, featuring clipboard copy functionality and real-time access analytics.


Click the link below to explore the full project and discover all its features:

[![Acesse o Projeto](https://img.shields.io/static/v1?label=Acesse%20o%20Projeto&message=Aqui&color=B20000&labelColor=000000)](https://ourpokedex.netlify.app)

### 🚀 Tech Stack

This project was developed with the following technologies:

#### Frontend
- Typescript & Tailwind
- Vite
- React
- Axios & React DOM
- Git & Github

### 🚧 Features

- [x] Create shortened links
    - [x] It must NOT be possible to create a link with malformed shortening
    - [x] It must NOT be possible to create a link with a shortening that already exists
- [x] Delete links
- [x] Redirect from short URL to original
- [x] List all registered URLs
- [x] Increment access counter
- [x] Download a CSV with the report of created links

### 💡 Quick Start

```bash
# 1. Clone & Install
git clone <repo> brevly
cd brevly/web
```

**Local Development**

`npm install` Install dependencies

`npm run dev` Start development server

### 💻 About

**Brev.ly** is a responsive URL shortener with a user dashboard to manage created links.

It shortens URLs with persistent storage, featuring clipboard copy functionality and real-time access analytics.


Click the link below to explore the full project and discover all its features:

### 🚀 Tech Stack

This project was developed with the following technologies:

#### Frontend
- Typescript & Tailwind
- Vite
- React
- Axios & React DOM
- Git & Github

#### Backend
- Fastify & Zod
- PostgreSQL
- Drizzle ORM
- Scalar UI
- AWS ECR & CDN (Cloudflare R2 )
- Git & Github

#### DevOps
- Docker
- Docker compose
- GitHub Actions (CI/CD)

### 💡 Quick Start

**Backend**
```bash
# 1. Clone & Install
git clone <repo> brevly
cd brevly/server
```

```bash
# 2. Start with Docker (Recommended)
docker-compose up -d
```

**Local Development**

`npm install` Install dependencies

`npm run db:migrate` Apply migrations

`npm run dev` Start development server

**Frontend**
```bash
# 3. Frontend
cd ../web
```

**Local Development**

`npm install` Install dependencies

`npm run dev` Start development server
